# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2026-03-11

### Changed
- Renamed `GENERIC-HYBRID-WORKFLOW.md` to `ATOMIC-TASK-WORKFLOW.md` for clarity
- Expanded atomic task definition with comprehensive examples
- Added visual workflow diagram
- Enhanced approval command documentation
- Improved verification standards and error handling guidance

### Removed
- `GENERIC-HYBRID-WORKFLOW.md` (replaced by `ATOMIC-TASK-WORKFLOW.md`)

## [1.0.0] - 2026-03-11

### Added
- Initial release
- `new-task` skill for planning phase
- `execute-task` skill for execution phase
- Task document templates (intake, plan, session)
- Atomic Task Workflow documentation
- CLI with `init` command
- Support for GitHub Copilot, Cursor, and Claude
